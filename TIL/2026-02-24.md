# 2026-02-24 TIL

## SOP (Same Origin Policy)

- 외부 출처(Protocol/Scheme, Host, Port)에서 불러온 데이터를 읽지 못하게 하는 브라우저의 보안 메커니즘(쓸 수는 있다!)

## CORS (Cross Origin Resource Sharing)

- HTTP 헤더 기반 Cross Origin 간 리소스 공유 방법

## XSS (Cross-Site Scripting)

- Client Side
- 공격자가 웹 리소스에 악성 코드를 삽입해 이용자의 웹 브라우저에서 해당 스크립트 실행하는 취약점
- Stored XSS: 서버에 존재하는 악성 스크립트를 이용자가 조회할 때 발생
- Reflected XSS: 이용자가 악성 스크립트가 포함된 요청을 보낸 후 응답을 출력할 때 발생
