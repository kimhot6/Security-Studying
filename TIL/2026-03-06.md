## JS

js는 웹 브라우저에서 작동하는 유일한 프로그래밍 언어

callback 함수: 다른 함수에 인자로 전달되어 사용되는 함수

### DOM(Document Object Model)

HTML, XML 문서의 구조화된 표현. 브라우저가 웹 페이지를 로딩할 때 문서를 트리 형태의 객체로 반환->이것이 DOM

- 웹 페이지 내 요소들은 DOM 트리의 **노드(Node)**로 표현된다
- JavaScript는 이 DOM에 접근하고 조작함으로써 동적인 콘텐츠 변경, 이벤트 처리, 인터페이스 제어 등이 가능

### 해커가 보는 DOM

**document.cookie (성배)**: DOM 구조의 최상단에 있는 보물상자. 이걸 읽어내는 게 웹 해킹의 1차 목표

**innerHTML (주사기)**: 웹페이지의 특정 공간에 내 HTML/JS 코드를 강제로 쑤셔 넣는 속성 (여기서 DOM Based XSS가 터짐)

**location.href (도주 차량)**: 쿠키를 훔친 뒤, 해커의 서버로 페이지를 이동시켜 버릴 때 쓰는 속성
