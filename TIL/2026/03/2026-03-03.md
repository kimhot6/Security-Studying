# DBMS

- 데이터베이스를 관리하는 애플리케이션

## Relational DBMS(RDBMS)

- 행과 열의 집합으로 구성된 테이블의 묶음 형식으로 데이터 관리
- 테이블 형식의 데이터 조작할 수 있는 관계 연산자 제공

## SQL(Structured Query Language)

- RDBMS의 데이터를 정의, 질의, 수정하기 위해 고안된 언어
- 구조화된 형태를 가지는 언어로 웹 애플리케이션이 DBMS와 상호작용할 때 사용됨
- 사용 목적에 따라 DDL, DML, DCL 등의 구조로 구분됨
