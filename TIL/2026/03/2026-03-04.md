# 2026-03-04 TIL

## NoSQL Injection

- **NoSQL**: SQL을 사용하지 않고 복잡하지 않은 데이터를 저장해 단순 검색 및 추가 검색 작업을 위해 최적화된 저장 공간
- **Collection**: DB의 하위에 속하는 개념으로 RDBMS의 테이블과 비슷함

## Command Injection

- **Command Injection**: 시스템 명령어에 대한 injection
- **메타 문자**: ; ` || && 등 특수한 의미를 가진 문자

## File Vulnerability

- **CGI**: 웹 페이지에서 동적인 컨텐츠를 처리하기 위해 웹 서버와 php같은 외부의 프로그램 사이에서 인터페이스를 제공하는 프로그램
- /proc/self/maps: Linux 운영체제에서 현재 실행중인 프로세스의 메모리 구조와 맵핑 정보를 담고 있는 특수 파일

## SSRF(Server-Side Request Forgery)

- **SSRF**: 웹 서비스의 요청을 변조하는 취약점으로, 브라우저가 변조된 요청을 보내는 CSRF와는 다르게 웹 서비스의 권한으로 변조된 요청을 보낼 수 있다
- **마이크로서비스**: 소프트웨어가 잘 정의된 API를 통해 통신하는 소규모의 독립적인 서비스로 구성되어 있는 경우의 소프트웨어 개발을 위한 아키텍처 및 조직적 접근 방식
- **백오피스 서비스(관리자 페이지)**: 관리자만이 수행할 수 있는 모든 기능을 구현한 서비스. 내부망에 위치
